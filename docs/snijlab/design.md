# Creating a design for laser cutting

## Workflow

1. Develop your concept (sketch, photo, idea)
2. Create a vector drawing in a supported program
3. Apply correct line colors and layer names (see [specs](specs.md))
4. Export as DXF, DWG, AI or PDF
5. Upload to Snijlab

## Supported software

* Adobe Illustrator
* AutoCAD
* Rhinoceros
* Inkscape
* Solidworks
* Onshape
* SketchUp
* Vectorworks

## Design rules

* Design at full size (1:1 scale), units in mm
* Maximum sheet size: 1200 x 600 mm
* Convert all text to vector outlines
* Remove unnecessary lines (dimensions, frames, centerlines)
* Use correct layer names and line colors (see [specs](specs.md))

## Box generators

* [MakerCase](https://www.makercase.com/) — use 0.2 mm burn offset
* [Boxes.py](https://www.festi.info/boxes.py/) — use 0.2 mm burn offset

## Templates

Snijlab provides pre-made drawing templates that can be modified for common shapes and joints.
