# Bamboo plywood

Cross-glued bamboo plywood, FSC 100% certified. Stronger than side-pressed bamboo due to plywood construction. Resists breaking with thin or fragile design elements.

## Properties

| Property        | Value                                    |
|-----------------|------------------------------------------|
| Color           | Warm caramel tone (heat-treated)         |
| Cut edges       | Dark brown                               |
| Engraved areas  | Medium brown                             |
| Grain           | Cross-glued layers, strong in all directions |
| Moisture        | Low, prevents warping                    |
| Max sheet size  | 1200 x 600 mm                            |
